# シナリオ03：コメントの管理と削除制約の確認

このシナリオでは、コメントの投稿、自身のコメント削除時の文言置換、およびコメントが存在する場合のスレッド削除不可ルールをテストします。

## 1. コメント投稿と削除のテスト
1. `testuser_01` でログインする。
2. 新しくスレッド「コメントテスト用」を作成する。
3. 作成したスレッド詳細画面で、本文に「テストコメント1」と入力して送信する。
4. コメント一覧に正しく表示されることを確認する。
5. 送信したコメントの横にある「削除」ボタンをクリックする。
6. コメントの本文が「＜このコメントは削除されました＞」に書き換わっていることを確認する。

## 2. スレッド削除制約のテスト（新ルール）
1. 同じスレッドに、さらに「テストコメント2」を投稿する。
2. スレッド詳細画面を確認する。
3. すでにコメントが存在するため、作成者であっても「スレッドを削除する」ボタンが表示されていない、あるいは無効化されていることを確認する。
4. 直接削除API/アクションを実行しようとしても、エラー（コメントが存在するため削除できない旨）が返されることを確認する。

## 3. 他者による操作の制限
1. `testuser_02` でログインする。
2. `testuser_01` が作成したスレッドの「テストコメント2」を表示する。
3. 他者のコメントであるため、削除ボタンが表示されないことを確認する。
