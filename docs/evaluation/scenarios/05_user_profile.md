# シナリオ05：ユーザープロフィールの変更と反映

このシナリオでは、ユーザーの表示名変更が既存の投稿すべてに正しく波及することをテストします。

## 1. 表示名変更のテスト
1. `testuser_01`（表示名：テストユーザー01）でログインする。
2. あらかじめスレッドとコメントをいくつか投稿しておく。
3. プロフィール編集画面（ユーザー編集画面）へ移動する。
4. 表示名を `変更後のユーザー名` に書き換えて保存する。
5. 16文字以上の名前を入力して保存を試み、エラーになることを確認する。
6. 正しい名前（変更後のユーザー名）で保存し、保存完了のメッセージが表示されることを確認する。

## 2. 変更内容の反映確認
1. スレッド一覧画面へ移動し、自分が作成したスレッドの作成者名が `変更後のユーザー名` になっていることを確認する。
2. スレッド詳細画面へ移動し、自分が投稿したコメントの作成者名も `変更後のユーザー名` になっていることを確認する。
3. ログアウトして再度ログインし、ログイン後も新しい表示名が維持されていることを確認する。
