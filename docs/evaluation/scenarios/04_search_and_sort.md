# シナリオ04：検索機能とソート機能のテスト

このシナリオでは、本文およびコメントを対象としたキーワード検索、およびソート機能が意図通りに動作するかをテストします。

## 1. 検索機能とスクロールのテスト
1. 下記の内容を含む複数のスレッドとコメントを作成しておく。
    - スレッドA 本文: `プログラミング言語Rustの特徴`
    - スレッドB コメント: `メモリ安全性が高いのがRustの魅力です`
2. 検索窓に `Rust` と入力して検索を実行する。
3. 検索結果一覧に「スレッドA」と「スレッドB」の両方が表示されていることを確認する。
4. 検索結果の「スレッドB」をクリックして詳細画面へ遷移する。
5. ヒットしたコメント（メモリ安全性が〜）が画面の表示領域内に表示されるまで、自動的にスクロールが行われることを確認する。

## 2. ソート機能のテスト
1. スレッド一覧画面へ移動する。
2. ソート条件を「作成日時（昇順）」に切り替える。
3. 一番古いスレッドがトップに表示されていることを確認する。
4. ソート条件を「コメント数」に切り替える。
5. コメント数が多いスレッド順に並んでいることを確認する。
6. ソート条件を「最終更新日時」に切り替える。
7. 直前にコメントを投稿した「スレッドB」が一番上に表示されていることを確認する。

## 3. ページネーションのテスト
1. 合計で11件以上のスレッドを作成する。
2. 1ページ目に10件のスレッドが表示され、「前に戻る」が表示されず「次に進む」が表示されていることを確認する。
3. 「次に進む」をクリックする。
4. 2ページ目に遷移し、残りのスレッドが表示され、「前に戻る」が有効になっていることを確認する。
