# コメントの作成機能

スレッドに対して返信や意見を述べるためのコメントを作成します。

## 詳細要件

- **入力項目**: 本文。
- **編集制限**: コメント作成後、本文の編集は一切不可となります。
- ログイン中のユーザーのみが作成可能です。

## 受け入れ基準 (Acceptance Criteria)

- [ ] スレッド詳細画面から、ログイン状態で本文を入力し、コメントを投稿できること。
- [ ] 本文が空の状態で投稿しようとした場合、エラーが表示されること。
- [ ] 投稿後、即座に詳細表示のコメント一覧に反映されること。
- [ ] コメント作成後に本文を編集するためのUI（ボタン等）が存在しないこと。
