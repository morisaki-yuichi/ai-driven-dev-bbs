# スレッドの削除機能

作成したスレッドを削除します。ただし、掲示板の整合性を保つため特定の制約があります。

## 詳細要件

- **削除権限**: スレッドの作成者本人のみが削除可能です。
- **削除制限**: すでにコメントがついているスレッドは、作成者であっても削除できません。
- **不可逆性**: 削除したスレッドは復元できません。

## 受け入れ基準 (Acceptance Criteria)

- [ ] スレッド作成者が、コメントが1件もついていない自分のスレッドを削除できること。
- [ ] すでに1件以上のコメント（削除されたコメントも含む）がついている場合、削除ボタンが表示されないか、エラーメッセージが表示されて削除できないこと。
- [ ] 自分以外のユーザーが作成したスレッドを削除できないこと。
- [ ] 削除が完了した後、スレッド一覧および詳細からそのスレッドが消えていること。
