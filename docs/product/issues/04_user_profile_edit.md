# ユーザーの編集機能

ログイン中のユーザーが自身のプロフィール情報を変更できるようにします。

## 詳細要件

- **編集可能項目**: 表示名のみ編集可能です。
- **制約**: 表示名は15文字以内である必要があります。
- その他の情報（ユニークID、パスワードなど）の変更機能は不要です。

## 受け入れ基準 (Acceptance Criteria)

- [ ] プロフィール編集画面で新しい表示名を入力し、保存できること。
- [ ] 表示名を変更した後、過去の投稿（スレッド、コメント）の表示名も新しい名前に反映されること。
- [ ] 15文字を超える表示名を入力して保存しようとした場合、バリデーションエラーが表示され保存できないこと。
