# スレッド詳細表示

スレッドの全内容と、それに紐づくコメントの一覧を表示します。

## 詳細要件

- **スレッド部表示内容**: タイトル、本文、作成日時、作成者。
- **コメント一覧部表示内容**:
    - 各コメントの本文（削除済みは専用メッセージ）
    - 作成日時
    - 作成者
- 一覧表示から特定のターゲットを選択した場合、詳細画面に遷移します。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ログインしていない状態でアクセスした場合、ログイン画面にリダイレクトされること。
- [ ] スレッド一覧から対象を選択すると、そのスレッドの全本文と関連する全コメントが表示されること。
- [ ] 削除されたコメントは「＜このコメントは削除されました＞」と表示され、本来の作成者や日時は維持されていること（または仕様に合わせて適切に表示されていること）。
- [ ] ログイン状態に応じて、自分のスレッドには「削除」ボタン（コメント0件の場合のみ）、自分のコメントには「削除」ボタンが表示されること。
- [ ] 削除されたスレッド、または存在しないスレッドのURLに直接アクセスした場合、エラーページ（404相当）が表示されること。
