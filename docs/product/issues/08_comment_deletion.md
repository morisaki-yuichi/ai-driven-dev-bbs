# コメントの削除機能

投稿したコメントを個別に削除します。

## 詳細要件

- **削除権限**: コメントの作成者本人のみが削除可能です。
- **削除後の表示**: データベース上から完全に消し去ることはせず、スレッドの整合性と文脈を維持するため、削除された箇所の本文には「＜このコメントは削除されました＞」という固定メッセージを表示する必要があります。
- **不可逆性**: 一度削除したコメントは復元できません。

## 受け入れ基準 (Acceptance Criteria)

- [ ] コメント作成者が、自分のコメントに対して削除操作を実行できること。
- [ ] 削除完了後、該当箇所の本文が「＜このコメントは削除されました＞」に置換されていること。
- [ ] 自分以外のユーザーが作成したコメントを削除できないこと。
- [ ] 削除されたコメントに対して、さらに「編集」や「再削除」などは行えないこと（表示のみ残る）。
