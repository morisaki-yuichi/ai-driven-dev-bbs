# スレッド一覧表示

投稿されたスレッドをメタデータとともに一覧表示します。

## 詳細要件

- **表示項目**:
    - タイトル
    - 本文（冒頭部分のみの省略表示可）
    - 作成日時
    - 作成者（表示名）
    - コメント数
    - 最終更新日時（スレッド作成時または最新のコメント投稿時の日時）
- **ページネーション**:
    - 1ページあたりの表示件数は10件。
    - 「前に戻る」「次に進む」のボタン/リンクが必須。
    - 1ページ目では「前に戻る」を、最終ページでは「次に進む」を表示しないこと。
    - その他（10件以上の表示切り替え、全件数表示、ページ番号ジャンプ）は任意。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ログインしていない状態でアクセスした場合、ログイン画面にリダイレクトされること。
- [ ] タイトル、本文、作成日時、作成者等の情報が正しく表示されていること。
- [ ] 11件以上のスレッドがある場合、正しくページネーションが機能し、11件目以降は2ページ目以降に表示されること。
- [ ] 最終更新日時は、コメントが投稿されるたびに更新されていること。
- [ ] ページ移動後、「前に戻る」「次に進む」が状況（最初・最後のページ）に応じて適切に制御されていること。
