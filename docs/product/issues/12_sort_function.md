# スレッドのソート機能

スレッド一覧の並び順をユーザーが切り替えられるようにします。

## 詳細要件

- **ソート条件**:
    1. **作成日時順**: スレッドが作成された日時の降順/昇順。
    2. **コメント数順**: スレッドに紐づくコメント数（削除済みを含む）が多い順。
    3. **最終更新日時順**: スレッドの作成、または新しいコメントが投稿された日時の降順。
- 初期表示は「作成日時順の降順」または「最終更新日時順の降順」を推奨します。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ログインしていない状態でアクセスした場合、ログイン画面にリダイレクトされること。
- [ ] ソート順を切り替える UI（セレクトボックス等）が存在すること。
- [ ] 「コメント数順」に変更した際、コメントが多いスレッドが上位に表示されること。
- [ ] 「最終更新日時順」に変更した際、最新のコメントがついたスレッドが一番上に表示されること。
- [ ] ページネーションと併用が正しく行われ、ページをめくってもソート順が維持されていること。
