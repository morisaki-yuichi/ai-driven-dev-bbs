# スレッドの作成機能

ユーザーが新しい話題を開始するためのスレッドを作成します。

## 詳細要件

- **入力項目**: タイトル、本文。
- **編集制限**: スレッド作成後、タイトルおよび本文の編集は一切不可となります。
- ログイン中のユーザーのみが作成可能です。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ログイン状態でタイトルと本文を入力し、スレッドを作成できること。
- [ ] タイトルまたは本文が空の状態で作成しようとした場合、エラーが表示されること。
- [ ] 作成後、スレッド一覧画面に新しいスレッドが表示されること。
- [ ] スレッド作成後にタイトルや本文を編集するためのUI（ボタン等）が存在しないこと。
