# 検索機能

スレッドの本文およびコメントの内容から、特定のキーワードを含む投稿を検索します。

## 詳細要件

- **検索対象**: 全スレッドの本文、および全コメントの本文。
- **検索結果**: 検索キーワードを含む「スレッド一覧」を表示します。
- **スクロール連携**: 検索結果から特定のスレッド詳細に遷移した際、キーワードが含まれる該当箇所（スレッド本文またはコメント）まで自動的にスクロールする必要があります。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ログインしていない状態でアクセスした場合、ログイン画面にリダイレクトされること。
- [ ] 検索窓にキーワードを入力して実行すると、本文またはコメントにその文字が含まれるスレッドが一覧表示されること。
- [ ] 検索結果からスレッド詳細へ遷移し、ヒットした箇所（コメント等）が画面内に表示されるまで自動的にスクロールされること。
- [ ] 削除済みのコメント本文「＜このコメントは削除されました＞」は検索対象にならない（あるいはヒットしても遷移時に矛盾がない）ように制御されていること。
