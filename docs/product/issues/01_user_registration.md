# ユーザー登録機能

ユーザーが新しくシステムを利用できるようにするためのアカウント作成機能を提供します。

## 詳細要件

- **ユニークID**: ユーザーを一意に識別するためのID。他ユーザーとの重複は不可。
- **パスワード**: セキュリティ確保のため、以下の強度要件を満たす必要があります。
    - 12文字以上
    - 英字、数字、記号をすべて含む
    - 使用可能な記号: `!@#$%^&*()_+-=[]{}|;':",./<>?`
- **表示名**: サイト内で表示される名前。
    - 15文字以内
    - 重複可能
- **救済機能**: パスワードを忘れた場合の救済機能は、本教材のスコープ外として実装不要です。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ユニークID、パスワード、表示名を入力して登録ボタンを押すと、アカウントが作成されること。
- [ ] すでに存在する名前のユニークIDで登録しようとした場合、エラーが表示されること。
- [ ] パスワードが強度要件（12文字以上、英数字記号の混在、許可された記号のみ）を満たさない場合、適切なバリデーションエラーが表示されること。
- [ ] 表示名が15文字を超える場合、登録できないこと。
- [ ] 登録成功後、ログイン画面へリダイレクトされること。
- [ ] ユーザー情報が永続化され、登録した情報でログイン可能な状態になっていること。
