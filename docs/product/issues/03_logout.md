# ログアウト機能

現在のユーザーセッションを破棄し、認証が必要な機能へのアクセスを制限します。

## 詳細要件

- ログアウト後は、ログインが必要な機能（スレッド作成、編集、削除など）が利用不可になります。

## 受け入れ基準 (Acceptance Criteria)

- [ ] ログアウトボタンを押すと、即座にログアウト処理が行われ、ログイン画面にリダイレクトされること。
- [ ] ログアウト後、ブラウザの「戻る」ボタンを押しても、ログイン画面にリダイレクトされること。
- [ ] ログアウト状態で認証が必要なURLに直接アクセスした場合、ログイン画面へリダイレクトされること。
